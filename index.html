REVISÃO

HTML

Criação de arquivo HTML, utilização do comando (html:5) para criação do escopo do documento, alteração do idioma, dentro do comando head, criamos o comando link para relacionar com o arquivo css e script para relacionar com o arquivo JS a ser executado.

SCRIPT.JS

Criação de arquivo JavaScrip, declarando variáveis que receberão valor, realizando uma condição (if) para confirmar que os dados recebidos estão corretos, se todos estiverem corretos emitimos um alerta dizendo que o cadastro foi concluído, caso algum dado seja inserido incorretamente emitimos um alerta para que o cadastro seja feito novamente.

STYLE

Criação de arquivo CSS para edição de estilos de plano de fundo da pagina web, fonte e cor dos textos utilizados no arquivo HTML.
